# 1. Vision

Bhavna Institute IMS is a centralized platform designed to manage the complete operations of Bhavna Institute from a single system.

The platform will replace manual registers, spreadsheets, and disconnected software tools by providing a unified solution for admissions, student management, attendance, fee collection, academics, reporting, and communication.

The initial goal is to digitize Bhavna Institute's operations. Future versions may evolve into a SaaS platform for multiple institutes.

# 5. Functional Requirements

## Student Management

### Create Student

The system must allow authorized users to create student records.

### Edit Student

Users must be able to update student information.

### Student Profile

The system must provide a detailed student profile page.

### Student Status

Student status options:

* Active
* Completed
* Dropped
* Inactive
* Alumni

---

## Course Management

The system must support:

* Course creation
* Course editing
* Course fee configuration

---

## Batch Management

The system must support:

* Batch creation
* Student assignment
* Trainer assignment
* Batch capacity management

---

## Attendance Management

The system must support:

* Daily attendance marking
* Attendance reports
* Attendance history

---

## Fee Management

The system must support:

* Fee plans
* Payment recording
* Receipt generation
* Pending fee tracking
