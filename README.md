# Bhavna Institute IMS

Institute Management System for Bhavna Institute - digitizing admissions, student management, attendance, fee collection, and reporting.

## Features

- **Dashboard** - Institute overview with stats and quick actions
- **Student Management** - Registration, profiles, enrollment, status tracking
- **Course Management** - Course creation, catalog, fee structure
- **Batch Management** - Batch creation, schedules, student assignment
- **Fee Management** - Fee plans, payment collection, receipts, pending tracking
- **Attendance** - Daily attendance, monthly reports, statistics
- **Lead CRM** - Lead registration, follow-ups, conversion to student
- **Staff Management** - Staff records and roles
- **Reports** - Student, admission, fee, and attendance reports
- **Settings** - Institute profile, user management, permissions
